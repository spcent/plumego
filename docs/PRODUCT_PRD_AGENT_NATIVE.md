# Archived Draft: Agent-Native Product PRD

This file is kept only as a historical pointer.

The original draft PRD was written while Plumego was still converging on its
agent-first repository operating model. Its detailed assumptions are no longer
the current authority and many planning items have already been implemented,
renamed, or folded into other control surfaces.

## Current Authority

Use these documents instead:

- `docs/ROADMAP.md` — active staged roadmap
- `docs/CODEX_WORKFLOW.md` — milestone execution workflow
- `docs/MILESTONE_PIPELINE.md` — artifact contract for milestone work
- `docs/architecture/AGENT_FIRST_REPO_BLUEPRINT.md` — repository shape and canonical path
- `specs/*` — machine-readable routing, taxonomy, dependency, and validation rules

## Why This Draft Was Archived

- it remained marked draft after the workflow had already moved into `tasks/*`, `specs/*`, and `internal/checks/*`
- several gaps described in the draft are now closed or materially different
- the active source of truth is now split on purpose between prose (`docs/`), machine-readable rules (`specs/`), and execution units (`tasks/`)

## Historical Note

Keep this path for backward references from archived task cards and historical
planning notes, but do not treat it as a current product contract.
