# M-013: Migration Guides

- **Branch:** `milestone/M-013-migration-guides`
- **Depends on:** M-008
- **Parallel OK:** yes

## Goal

Publish docs/migration/ with four guides (from-gin, from-echo, from-chi,
middleware-compat) that give teams on existing frameworks a clear, code-anchored
path to adopting plumego incrementally.

## Architecture Decisions

- Guides live in docs/migration/; no code changes to stable roots or x/*.
- Each guide includes: a mapping table of framework concepts to plumego
  equivalents, the three most common handler/middleware pattern conversions as
  runnable Go snippets, and an explicit list of incompatibilities.
- Code snippets must compile against the current API surface; they are checked
  by a go vet pass on docs/migration/examples/ if examples are added.
- No compatibility shims or wrappers are introduced; the guide explains how to
  translate patterns, not how to run both frameworks simultaneously.
