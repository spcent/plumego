# M-008: v1.1.0 Release Execution

- **Branch:** `milestone/M-008-v1-1-0-release`
- **Depends on:** M-007
- **Parallel OK:** no

## Goal

Tag v1.1.0, produce release evidence, and provide the second release ref that
unblocks all five blocked beta-evidence closure cards (1367, 1370, 1371, 1372, 1373).

## Architecture Decisions

- v1.1.0 must show zero stable-root public API changes vs v1.0.0; snapshot
  comparison is required before tagging.
- No new external dependencies may be introduced before the tag.
- The second_release_ref in specs/extension-beta-evidence.yaml must be updated
  to v1.1.0 only after the tag exists; do not use HEAD as a release ref.
- Release notes live in docs/release/v1.1.0.md following the structure of
  docs/release/v1.0.0.md.
- Blocked cards 1367, 1370, 1371, 1372, 1373 are updated in the same PR; they
  move from blocked/ to active/ only if all evidence blockers are cleared.
