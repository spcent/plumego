# M-011: Benchmark Suite Publication

- **Branch:** `milestone/M-011-benchmark-suite`
- **Depends on:** M-008
- **Parallel OK:** yes

## Goal

Publish a benchmark suite covering router matching and middleware chain
throughput, with CI-captured results committed to docs/benchmarks/ for
community trust and framework positioning.

## Architecture Decisions

- Benchmarks live in benchmark/ at the repo root as a standalone package.
- No new external dependencies; use only stdlib testing/benchmark primitives
  and net/http/httptest.
- Comparisons include chi (github.com/go-chi/chi) at a pinned version in a
  separate benchmark/go.mod to keep the main module stdlib-only.
- Results snapshot is committed as docs/benchmarks/results-v1.1.0.md with
  the exact go test -bench output pasted verbatim.
- README.md and docs/why-plumego.md reference the results file by path only;
  no inline numbers that will rot.
