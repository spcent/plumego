## 已知问题与限制

### 关键问题（需修复）

1. **真实数据库集成覆盖不足**
   - Redis/MongoDB/Elasticsearch 已接入 active operation registry
   - 仍需在 Docker 多数据源环境补充端到端取消验证

2. **大数据完整内容访问不完整**
   - BLOB/TEXT/JSON 预览已有限制
   - BLOB 下载、按需加载完整大字段、统一响应体上限仍需补齐

3. **连接生命周期管理**
   - 设置页可查看运行时连接计数并关闭指定连接运行时
   - 主工作区资源树尚未展示连接健康状态

### 次要问题

5. **错误码不统一**
   - 部分处理器使用 contract.ErrorType
   - 部分处理器返回自定义错误
   - 前端错误展示不一致

6. **功能缺失**
   - 无统一的响应大小限制
