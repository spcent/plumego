## 实际判断规则

在把某个包视为长期可靠依赖之前，先判断它属于稳定根，还是属于仍在演进的扩展家族。

## 为什么要单独写这一页

Plumego 把发布姿态写清楚，是为了避免文档、脚手架和扩展工作无意中夸大兼容性承诺。
