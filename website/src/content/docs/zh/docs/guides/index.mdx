# 操作指南

这些指南是实用配方。每篇针对单一任务，展示完成任务所需的最少代码，并在需要边界原理时链接到相关的模块 primer。

阅读前提：已完成 [Getting Started](/zh/docs/getting-started) 并能运行参考服务。

## 指南的范围说明

每篇指南展示一种正确模式并就此停止。它们不尝试覆盖每个选项或配置旋钮 — 模块 primer 和源码树中的 `module.yaml` 文件负责那部分内容。如果指南代码与模块的 `module.yaml` 有出入，`module.yaml` 是权威来源。

稳定根指南不会将 `x/*` 扩展引入规范服务路径。运维指南在该能力的正确归属是 `x/observability` 时会引用它——每次引用都链接到扩展 primer 以说明边界原理。
