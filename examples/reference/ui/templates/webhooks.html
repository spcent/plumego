Security Notes

Keep tokens and signatures safe and avoid logging secrets.

  • Set WEBHOOK_TRIGGER_TOKEN in env for event triggers.
  • GitHub/Stripe signatures are verified in webhook ingress.
  • Rotate secrets regularly for production use.
